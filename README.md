🧮 Smart Calculator

A modern and responsive calculator built using HTML, CSS, and JavaScript with a clean UI and smooth user experience.

---

🚀 Features

- ➕ Basic arithmetic operations (+, −, ×, ÷)
- 📊 Percentage calculations
- 🔢 Decimal support
- ⌫ Backspace functionality
- 🧹 Clear (reset) option
- ⌨️ Keyboard support
- 🎨 Modern UI with animations
- 📱 Fully responsive design
- ⚠️ Error handling
